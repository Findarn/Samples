Генерация лабиринта, поиск выхода с помощью волнового алгоритма.

ООП, кастомный динамический стек, волновой алгоритм, генерация лабиринта.

Описать класс, реализующий стек. Написать программу, использующую этот класс для поиска прохода по лабиринту. 
Лабиринт представлен в виде матрицы, состоящей из квадратов. Каждый квадрат либо открыт, либо закрыт.
Вход в закрытый квадрат запрещен. Если квадрат открыт, то вход в него возможен со стороны, но не с угла.
Каждый квадрат определяется его координатами в матрице. После отыскания выхода программа печатает путь в виде координат.
